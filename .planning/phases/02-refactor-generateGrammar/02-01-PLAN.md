# Plan 02-01: 建立 generate_grammar 基础模块和数据入口

<objective>
在 `src/LoPS/generate_grammar/` 下建立 Phase 2 新架构的基础模块：配置对象、token 工具、状态依赖图读取、StrategySequence 数据读取和基础 unittest。该计划不实现 BD score 和 grammar 学习算法，只建立后续计划依赖的清晰接口。
</objective>

<context>
本计划落实 `02-DESIGN.md` 的模块设计、类设计和 token 表示设计。核心约束：新核心使用 `list[str]` 和 `"G-L"` 形式 token；原始项目路径只读；配置必须显式接收输入目录、状态图目录、输出目录和学习参数。
</context>

<threat_model>
## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| 新配置默认写入原项目 `grammar2/` | HIGH | `GenerateGrammarConfig` 默认输出目录必须是 `data/generate_grammar/refactored-output/grammar2/`。 |
| 读取 `fileNames` 时丢失旧字段精确值 | HIGH | 同时保留 `participant_file_names` 原始值和 `participant_ids` 去后缀值。 |
| token 工具退回字符串字符级算法 | MEDIUM | 所有复合 token 操作必须通过 `split_token()`、`combine_tokens()`、`token_length()` 和 `tokens_share_base_token()`。 |
</threat_model>

创建 `token.py`，实现以下函数：

```python
def split_token(token: str) -> list[str]
def format_token(base_tokens: Sequence[str]) -> str
def combine_tokens(parent_token: str, child_token: str) -> str
def token_length(token: str) -> int
def tokens_share_base_token(left: str, right: str) -> bool
```

行为必须是：
- `split_token("G") == ["G"]`
- `split_token("G-L-E-A") == ["G", "L", "E", "A"]`
- `format_token(["G", "L"]) == "G-L"`
- `combine_tokens("G-L", "E-A") == "G-L-E-A"`
- `token_length("G-L-E-A") == 4`
- `tokens_share_base_token("G-L", "L-E") is True`
- `tokens_share_base_token("G-L", "E-A") is False`
</action>

在 `state_graph.py` 中定义：

```python
@dataclass
class StateDependencyGraph:
    conditions_by_state: list[list[int]]

def load_state_dependency_graph(path: Path) -> StateDependencyGraph
```

`load_state_dependency_graph()` 必须读取旧 `StateGraph` pickle 中的 `G`，对每行执行 `np.where(G[i, :] == 1)[0]`，返回 `conditions_by_state`。

在 `data_io.py` 中定义：

```python
@dataclass
class StrategyStateData:
    input_file_name: str
    token_sequence: list[str]
    initial_tokens: list[str]
    state_features: pd.DataFrame
    participant_file_names: list[str]
    participant_ids: list[str]

def list_strategy_state_files(strategy_sequence_dir: Path) -> list[str]
def load_strategy_state_data(path: Path, state_names: Sequence[str]) -> StrategyStateData
def write_generate_grammar_output(output: Mapping[str, Any], path: Path) -> None
```

`load_strategy_state_data()` 必须：
1. 读取旧 pickle。
2. 将旧 `seq` 转为 `list[str]`。
3. 将旧 `S` 转为 `list[str]`。
4. 读取 `state[state_names]`。
5. 将旧 `fileNames` 原始值保存为 `participant_file_names`，例如 `031222-401.pkl`。
6. 将去掉 `.pkl` 后的值保存为 `participant_ids`，例如 `031222-401`。

`list_strategy_state_files()` 必须只返回 `.pkl` 文件名，并排序。
</action>
