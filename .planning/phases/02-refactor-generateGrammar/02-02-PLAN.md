<action>
在 `scoring.py` 中实现：

```python
def learn_state_condition_links(
    data: np.ndarray,
    nstates: np.ndarray,
    block_message: dict[int, list[int]],
    casual_num: int,
    block_num: int,
    effect_num: int,
    alpha: float,
    conditions: list[list[int]],
) -> tuple[np.ndarray, list, list, list]
```

行为必须复刻旧 `learnBayesNetBlock`：
1. `var_num = data.shape[0]`。
2. `var_casual = list(range(block_num))`。
3. `var_effect = list(range(var_num - effect_num, var_num))`。
4. `Alearn = np.zeros((block_num + effect_num, block_num + effect_num))`。
5. 对每个 effect `v` 和每个 parent block `pa`：
   - `condition = conditions[pa]`
   - `condition = [block_message[c] for c in condition]`
   - `condition = sum(condition, [])`
   - 用 `bd_score(data[v, :], data[condition, :], nstates[v], nstates[condition], alpha / (np.prod(nstates[v]) * np.prod(nstates[condition])))` 得到 `bd1`
   - `Pa = condition + block_message[pa]`
   - 用同样方式得到 `bd2`
   - 若 `bd1 / bd2 > 1`，设置 `Alearn[pa, v - (casual_num - block_num)] = 1`
6. 返回 `(Alearn, bestparameters, bestparents, bestscores)`，后三项保持旧函数兼容的空列表结构。
</action>
