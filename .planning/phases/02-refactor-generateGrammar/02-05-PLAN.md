<action>
创建 `script/validate_generate_grammar.py`，使用 `argparse` 支持：
`--strategy-sequence-dir`、`--state-graph-dir`、`--baseline-grammar-dir`、`--output-dir`、`--alpha`、`--max-iterations`。

实现函数：

```python
def compare_values(old_value: Any, new_value: Any, path: str) -> list[str]
def compare_legacy_dict(old: Mapping[str, Any], new: Mapping[str, Any], file_name: str) -> list[str]
def validate_outputs(config: GenerateGrammarConfig) -> int
```

比较规则必须是：
1. 旧 dict 中每个 key，新 `legacy` 中必须存在。
2. `np.ndarray` 用 `np.array_equal()`。
3. `pd.DataFrame` 用 `pd.testing.assert_frame_equal(..., check_exact=True)`。
4. list/tuple 逐项递归比较。
5. dict 逐 key 递归比较。
6. float/int/bool/str 用 `==` 精确比较。
7. 不使用数值容差。
8. 输出所有差异的文件名和 key path。

`validate_outputs()` 必须先调用 `run_generate_grammar(config)` 生成新输出，再逐文件读取旧基准和新输出的 `legacy` 字典。
</action>
